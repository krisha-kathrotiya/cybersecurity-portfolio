# Controls Assessment Checklist – Botium Toys

This section provides a simplified overview of which security controls are currently in place and which are missing, based on the audit findings.

## Controls Assessment

| Control | Status | Explanation |
|--------|--------|------------|
| Least Privilege | Not in place | All employees have full access to sensitive data instead of limited, role-based access. |
| Disaster Recovery Plans | Not in place | There is no plan to recover systems or data in case of failure. |
| Password Policies | In place (weak) | A password policy exists, but it is not strong enough to ensure security. |
| Separation of Duties | Not in place | Responsibilities are not divided properly, increasing the risk of misuse. |
| Firewall | In place | A firewall is implemented and helps filter network traffic. |
| Intrusion Detection System (IDS) | Not in place | There is no system to monitor or detect suspicious activity. |
| Backups | Not in place | Important data is not backed up, which is a major risk. |
| Antivirus Software | In place | Antivirus software is installed and actively used. |
| Manual Monitoring (Legacy Systems) | Partially in place | Systems are monitored manually, but there is no structured process. |
| Encryption | Not in place | Sensitive data is stored without encryption. |
| Password Management System | Not in place | There is no secure, centralized way to manage passwords. |
| Locks (Physical Security) | In place | Physical locks are installed to secure facilities. |
| CCTV Surveillance | In place | CCTV systems are operational and monitoring the environment. |
| Fire Detection Systems | In place | Fire alarms and safety systems are active and functional. |

## Simple Notes (Based on Evidence)

- Employees have unrestricted access to data, which shows that proper access control is missing  
- Sensitive data is not encrypted, making it vulnerable to exposure  
- There is no intrusion detection system, so threats may go unnoticed  
- No backups or recovery plans exist, creating a high risk of data loss  
- Basic protections like firewall and antivirus are already in place  
- Physical security controls are strong and functioning properly  

## Summary

Botium Toys has implemented some basic security controls, mainly at the network and physical level. However, several important controls related to data protection, monitoring, and recovery are missing.

To improve overall security, the organization needs to focus on strengthening access control, protecting sensitive data, and preparing for potential system failures.
